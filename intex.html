Estrutura básica do HTML :

O HTML é a base para organizar as informações que queremos exibir em nossas páginas da web. Vamos dar uma olhada na estrutura básica:

  1. < html >: Essa tag envolve todo o conteúdo da nossa página. É o que será mostrado aos nossos usuários (ou não).
  2. < head >: Aqui, colocamos todas as informações necessárias que precisam ser carregadas antes de exibi-las ao usuário
  3. < body >: É onde todo o conteúdo da página é apresentado aos usuários: imagens, listas e qualquer outro conteúdo necessário.

Agora, você deve estar se perguntando: “Por que ela colocou < head > no começo do parágrafo e no final?” Bem, é assim que as tags funcionam.
A tag < html > é aberta sempre no início e só é finalizada depois que fechamos o < / body>.
Dessa forma, todo o conteúdo fica dentro dessa tag.

Explicando sobre as tag


Na tag < head >, podemos incluir várias outras tags além da < title >, que é usada para especificar o título da página HTML. A tag < link > é usada para vincular folhas de estilo CSS e arquivos JavaScript, não a tag < a >, que é usada dentro da tag < body > para criar links. O < head > contém meta-informações sobre a página, como o título e os recursos necessários para carregar a página corretamente.

Na tag , podemos incluir muitos elementos diferentes para criar páginas da web incríveis, como:

  1. Links (âncoras): Usamos a tag < a > para adicionar links para páginas externas ou outras partes do nosso próprio site.
  2. Títulos (headings): Variam de < h1 > (maior e mais importante) até < h6 > (menor e menos relevante).
  3. Parágrafos: A tag < p > é usada para criar parágrafos, onde colocamos o texto principal da página.
  4. Imagens: A tag < img > permite inserir imagens, com o atributo src especificando o caminho para a imagem.
  5. Estilos de Texto: Temos < i > itálico, < strong > para negrito, < u > para sublinhado e < mark > para marcações no texto.
  6. Listas: Podemos criar listas ordenadas com < ol > e listas não ordenadas com < ul >, usando < li > para cada item da lista.
  7. Realmente, há muitas possibilidades dentro da tag < body >! E isso é apenas uma fração do que pode ser feito. Para saber mais, o site W3Schools é uma excelente fonte de conhecimento sobre HTML

Explicando atributo :

Atributos são como pequenos ajudantes das tags no HTML que dizem a elas como se comportar ou o que mostrar. Existem dois tipos: os genéricos, que podem ser usados em quase todas as tags, e os específicos, que só funcionam em algumas tags. Por exemplo, o atributo id é como um nome de identificação que damos para uma coisa, no HTML, usamos o id para dizer que aquela parte da página é única e podemos usar esse nome para conectar coisas, como o título no índice com a parte correspondente da página. Já o atributo src é como o endereço de onde mora uma imagem na internet. Ele é usado só na tag < img > para dizer onde a imagem está, para que ela possa aparecer na página.

Então, os atributos ajudam as tags a saber o que fazer, como um endereço ajuda as pessoas a encontrar sua casa.
